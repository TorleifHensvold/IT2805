Questions

  1. The server in the client-server architecture is responsible for answering to any and all clients trying to make contact, and deliver the resources the client requests of the server.
  2. The client receives the data/resources the server sends to the client based on the requests by the client
  3. An IP-address is a number that uniquely [1] identifies a computer connected to the Internet. It is used to direct Internet traffic to the correct computer.
  4. DNS is the Domain Name Service, also referred to as the backbone of the Internet. It allows resolving human-readable domain names to the IP-addresses that host the content of the fully-qualified domain name (FQDN).
  5. The difference between the Internet and WWW is that the Internet is the way we access and send information, while WWW (or the Web) is a service built on top of the Internet to allow for accessing a subset of the computers and information that is theoretically accessible through the Internet.
  6. HTML is an acronym for HyperText Markup Language, and it is simply a way to tell browsers how to display the contents of of a file, and to interconnect those files through hyperlinks.
[1] Since we have more computers (including IoT) than we have available IP-addresses in IPv4 we have used NATs to sidestep this limitation by consolidating several computers behind one interface with one IPv4 address and allow the NAT to direct the packets to the right computer on its internal network.
